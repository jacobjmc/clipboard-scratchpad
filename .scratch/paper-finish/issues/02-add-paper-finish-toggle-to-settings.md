# Add Paper Finish Toggle To Settings

Status: needs-triage

## Parent

.scratch/paper-finish/PRD.md

## What to build

Add a native Settings toggle for the persisted Paper Finish setting, placed directly under Appearance, so the user can turn the editor paper finish on or off.

## Acceptance criteria

- [ ] Settings shows a row titled `Paper Finish` directly under Appearance and before Global Shortcut.
- [ ] The row description is `Add a subtle matte texture to the note.`
- [ ] The row uses a native toggle control.
- [ ] Toggling the control updates the persisted Paper Finish setting.
- [ ] The setting value remains correct after relaunch.
- [ ] Settings does not add intensity controls, texture style choices, or explanatory meta copy.
